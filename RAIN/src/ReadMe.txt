The code contains the implementation of our role-aware information diffusion model. 
Please execute make to compile the source code. 
After that, execute ./main to run the code.

Input parameters:

-n  string  :   network file dir 
-d  string  :   post file dir
-p  string  :   PageRank score file dir
-s  string  :   NetworkConstraint socre file dir
-i  int     :   number of maximum iterations used for learning algorithm [optional, 10 as default]
-k  int     :   number of social roles [optional, 3 as default]
